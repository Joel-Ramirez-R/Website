Sales Monitoring Dashboard for a Company Specialized in Hydraulic and Solar Equipment

The objective of this project is to accurately measure the performance of each field sales representative, as well as their corresponding region and zone. The solution tracks key performance indicators such as accumulated sales, global sales targets, progress percentage, daily sales, monthly sales comparison, and average daily sales.

Project Overview

Previously, this analysis was performed manually by extracting data from SAP into Excel and generating month-to-month comparisons, which consumed 48 hours of manual work per month. The new automated approach eliminates this manual effort, ensures data consistency, and provides real-time visibility into sales performance across all levels of the organization. As part of this improvement, 87 sales representatives now receive and review their sales status on a weekly basis, allowing for timely insights and better decision-making.

Tools Used

  • Power BI for dashboard creation
  • SQL Server for data storage
  • Power Query & DAX for KPI calculations

Process

  1. Define objectives and requirements with the Sales and Human Resources Departments.
  2. Analyze and prepare the data.
  3. Build the data model Create the analytical model with relationships, measures, and business logic.
  4. Build dashboards, verify calculations, test with users, and make adjustments.
  5. Publish to production, train users, monitor performance, and update the project as needed.
